## <a id="жанр"></a>Жанр
- **Match-three** - жанр, в котором нужно формировать линии из трёх и более одинаковых элементов
- **RPG** - не полноценная RPG: игрок двигается только при сборе трёх элементов в ряд
- **Головоломки** - нахождение вариантов соединения одинаковых элементов в ряд

## <a id="синопсис"></a>Синопсис
- Освещение: использование керосиновой лампы в руках персонажа как оружия и элемента освещения
- Попытка добраться до дома

## <a id="управление"></a>Управление
- Взмахи или нажатия пальцем на телефоне
- Клавиша действия: <kbd>Левая кнопка мыши</kbd>
- Комбинации:
  - <kbd>Кл.д</kbd> + <kbd>Верх</kbd>
  - <kbd>Кл.д</kbd> + <kbd>Вниз</kbd>
  - <kbd>Кл.д</kbd> + <kbd>Влево</kbd>
  - <kbd>Кл.д</kbd> + <kbd>Вправо</kbd>
  - <kbd>Кл.д</kbd> + двойное нажатие на бонус "Призрак-бум"
  - <kbd>Кл.д</kbd> + двойное нажатие на бонус "Волк с петардой" (по горизонтали или вертикали)
- Счетчик ходов

## <a id="стиль-игры"></a>Стиль игры
- Ролевая система
- Управление множеством объектов
- Цель
- Мрачный, ближе к черно-белому

## <a id="геймплейный-стиль"></a>Геймплейный стиль
- **Кор-геймплей** - не совсем каждый раунд задумывается как новый набор объектов, но в целом это цикл: набор действий, которые игрок повторяет чаще всего на протяжении всей игры для цели выиграть
- Геймплей по ходу игры может измениться с добавлением новых правил

## <a id="передвижение"></a>Передвижение
- Персонаж движется только при выполненных действиях, но при ожидании ходит из одной стороны в другую, которая доступна
- Есть карта из энного числа уровней как в Inscryption
- Счетчик очков

## <a id="системы-и-механики"></a>Системы и механики
### Системы (правила, которые всегда работают)
- Комбинирование элементов
- Повторяемость
- Внимательность
- Прокачка предметов

### Механики (правила, которые иногда работают)
- Трата керосина
- Уничтожение нечести где ходит перс
- Нахождение элементов для комбинирования

## <a id="враги"></a>Враги
1. Могут отнять керосин
2. Нельзя двигать
3. Могут сами меняться после хода игрока

## <a id="враги_как_бонус"></a>враги-как-бонус
- ### Призрак бум  Взрывают ближайшие элементы
- ### волк с петардой бьют по горизантали на 2 клетки или вертикале при выборе играка

## <a id="Параметры"></a>Параметры
- ## Керасин ± 100 единиц
- ## трата на один уровень ± 12 единиц 
- ## количество ходов за уровень ± 28 
- ## ± 7 керасина за ± 3000 очков  50 за каждое  комбинирование элементов
